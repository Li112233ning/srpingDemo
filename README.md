# 大学生课设(公司员工部门管理系统)

## 采用前后端分离技术(注释写满，易懂)

### 技术栈：SpringBoot+mysql+mybatis+maven+vue

## 运行
  前端项目跑起来以后，打包放到nginx，
  后端直接idea运行
  ### 需要更改的配置
  application.yml里面的数据库和阿里云oss配置(不改会报错，我的key已经失效)
  运行端口http://localhost:90/

员工的增删改查，
事务管理，JWT令牌校验，文件上传(要改oss配置)
